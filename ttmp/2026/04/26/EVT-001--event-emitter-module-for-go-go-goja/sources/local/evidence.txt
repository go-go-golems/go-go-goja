# go-go-goja event-emitter evidence

## Repository runtime/module architecture
     1	# go-go-goja – Native-Module Playground for goja
     2	
     3	**go-go-goja** is a tiny sandbox project that shows how to wire Go-implemented "native" modules into a [dop251/goja] JavaScript runtime using the [goja_nodejs/require] subsystem.
     4	
     5	The goal is to have a place where you can:
     6	
     7	* add your own Go files under `modules/` and immediately use them from JS via `require("your-module")`
     8	* experiment with goja + Node-style `require()` without having to set up a whole application
     9	* open the canonical interactive prompt with `go run ./cmd/goja-repl tui`
    10	* inspect and persist REPL sessions through the `goja-repl` CLI and JSON server surfaces
    11	* compose runtime behavior explicitly via `engine.NewBuilder() -> Build() -> Factory.NewRuntime(...)`
    12	
    13	---
    14	
    15	## Folder layout
    16	
    17	```text
    18	 go-go-goja/
    19	 ├── cmd/
    20	 │   ├── goja-repl/       # canonical REPL CLI, TUI, and JSON server
    21	 │   └── bun-demo/        # bun-integrated demo command
    22	 ├── engine/              # builder/factory/runtime ownership APIs
    23	 ├── modules/             # ← add your Go-backed modules here
    24	 │   ├── common.go        # registry plumbing (NativeModule, Register, …)
    25	 │   ├── fs/              # example module 1: basic file-system helpers
    26	 │   └── exec/            # example module 2: thin wrapper around os/exec
    27	 ├── testdata/            # demo JS scripts used by Go tests
    28	 ├── repl_test.go         # Go test that runs a JS script through the runner
    29	 └── go.mod
    30	```
    31	
    32	The canonical API is explicit runtime composition:
    33	
    34	1. create a builder (`engine.NewBuilder(...)`)
    35	2. add module/runtime options (`WithModules(...)`, `WithRuntimeInitializers(...)`, ...)
    36	3. build an immutable factory (`Build()`)
    37	4. create an owned runtime (`factory.NewRuntime(ctx)`)
    38	5. close runtime explicitly (`rt.Close(ctx)`)
    39	
    40	Legacy convenience wrappers (`engine.New()`, `engine.NewWithOptions(...)`, `engine.Open(...)`) were removed.
    41	
    42	---
    43

    47	# from the project root

    49	
    50	# open the canonical prompt (type JS, :quit to exit)
    51	❯ go run ./cmd/goja-repl tui --log-level debug

    55	hi

    60	### Runtime API quick example (current)
    61	
    62	```go
    63	ctx := context.Background()
    64	
    65	factory, err := engine.NewBuilder().
    66	    WithModules(engine.DefaultRegistryModules()).
    67	    Build()
    68	if err != nil {
    69	    return err
    70	}
    71	
    72	rt, err := factory.NewRuntime(ctx)
    73	if err != nil {
    74	    return err
    75	}
    76	defer rt.Close(ctx)
    77	
    78	vm := rt.VM
    79	_, err = vm.RunString(`console.log("hello from go-go-goja")`)
    80	if err != nil {
    81	    return err
    82	}
    83	```

    85	Notes:
    86	- `DefaultRegistryModules()` enables modules that registered themselves through `modules.Register(...)`.
    87	- `rt` bundles `VM`, `Require`, `Loop`, and `Owner` for explicit lifecycle control.

   107	### Script Module-Root Helper
   108	
   109	When your JS entrypoint lives in a nested folder (for example `js/extractor/main.js`),
   110	you can derive the standard `require.WithGlobalFolders(...)` layering from script path:
   111	
   112	```go
   113	factory, err := engine.NewBuilder(
   114	    engine.WithModuleRootsFromScript(
   115	        "/abs/path/to/js/extractor/main.js",
   116	        engine.DefaultModuleRootsOptions(),
   117	    ),
   118	).Build()
   119	```

   179	### Tips & conventions
   180	
   181	* Always keep your module self-contained inside `modules/<name>/<name>.go` – easier to copy around.
   182	* Return only **plain Go types** (string, number, bool, maps, slices) or `goja.Value`s. The runtime converts between Go & JS automatically.
   183	* If your module allocates goroutines, honour `context.Context` and consider `errgroup` for clean cancellation.
   184	* Use `var _ modules.NativeModule = (*yourType)(nil)` for compile-time checks (see [Go guidelines in the repo rules]).
   185	* Prefer explicit composition in application setup (`NewBuilder`, `WithModules`, `Build`, `NewRuntime`) over hidden global setup patterns.
   227	## Asynchronous APIs (Promises & Callbacks)
   228	
   229	`goja` lets Go code create real JavaScript `Promise`s or invoke JS callbacks later, but all VM access must happen on the runtime owner thread.
   230	
   231	The recommended reusable pattern is `pkg/runtimeowner`:
   232	
   233	```go
   234	runner := runtimeowner.NewRunner(vm, loop, runtimeowner.Options{
   235	    Name:          "my-module",
   236	    RecoverPanics: true,
   237	})
   238	
   239	promise, resolve, reject := vm.NewPromise()
   240	go func() {
   241	    out, err := slowWork()
   242	    _ = runner.Post(context.Background(), "myModule.settle", func(context.Context, *goja.Runtime) {
   243	        if err != nil {
   244	            _ = reject(vm.ToValue(err.Error()))
   245	            return
   246	        }
   247	        _ = resolve(vm.ToValue(out))
   248	    })
   249	}()
   250	return vm.ToValue(promise)
   251	```
   252	
   253	Low-level `loop.RunOnLoop(...)` is still valid, but `runtimeowner.Runner` is preferred for:
   254	
   255	- cancellation-aware `Call`/`Post`,
   256	- standardized errors (`ErrClosed`, `ErrScheduleRejected`, etc.),
   257	- owner-context fast-path for nested calls.
   258	
   259	Important caveat:
   260	
   261	- do not run blocking synchronous flows on the owner thread if those flows wait on background work that itself schedules back onto owner, or you can deadlock.
   262	
   263	### Demo: `timer` module
   264	
   265	`go-go-goja` now ships a built-in `timer` module when you enable `DefaultRegistryModules()`:
   266	
   267	```js
   268	const { sleep } = require("timer");
   269	await sleep(1000);   // returns a Promise that resolves after 1 s
   270	console.log("done");
   271	```
   272	
   273	The Go side (simplified):
   274	
   275	```go
   276	exports.Set("sleep", func(ms int64) goja.Value {
   277	    p, resolve, _ := vm.NewPromise()
   278	    go func() {
   279	        time.Sleep(time.Duration(ms) * time.Millisecond)
   280	        _ = runner.Post(context.Background(), "timer.sleep.resolve", func(context.Context, *goja.Runtime) {
   281	            _ = resolve(goja.Undefined())
   282	        })
   283	    }()
   284	    return vm.ToValue(p)
   285	})
   286	```
   287	
   288	Use it as a concrete built-in and as a template for any async binding you need (HTTP fetchers, database calls, …).
   156	func (f *Factory) NewRuntime(ctx context.Context) (*Runtime, error) {
   157		if f == nil {
   158			return nil, fmt.Errorf("factory is nil")
   159		}
   160		if ctx == nil {
   161			ctx = context.Background()
   162		}
   163	
   164		vm := goja.New()
   165		loop := eventloop.NewEventLoop()
   166		go loop.Start()
   167	
   168		owner := runtimeowner.NewRunner(vm, loop, runtimeowner.Options{
   169			Name:          "go-go-goja-runtime",
   170			RecoverPanics: true,
   171		})
   172		// #nosec G118 -- the runtime owns this cancel func and calls it on close and on setup failures.
   173		runtimeCtx, runtimeCtxCancel := context.WithCancel(context.Background())
   174		runtimeValues := map[string]any{}
   175	
   176		rt := &Runtime{
   177			VM:               vm,
   178			Loop:             loop,
   179			Owner:            owner,
   180			Values:           runtimeValues,
   181			runtimeCtx:       runtimeCtx,
   182			runtimeCtxCancel: runtimeCtxCancel,
   183		}
   184	
   185		runtimebridge.Store(vm, runtimebridge.Bindings{
   186			Context: runtimeCtx,
   187			Loop:    loop,
   188			Owner:   owner,
   189		})
   190	
   191		reg := require.NewRegistry(f.settings.requireOptions...)
   192		if err := DataOnlyDefaultRegistryModules().Register(reg); err != nil {
   193			_ = rt.Close(ctx)
   194			return nil, fmt.Errorf("register data-only default modules: %w", err)
   195		}
   196		for _, mod := range f.modules {
   197			if err := mod.Register(reg); err != nil {
   198				_ = rt.Close(ctx)
   199				return nil, fmt.Errorf("register module %q: %w", mod.ID(), err)
   200			}
   201		}
   202		moduleCtx := &RuntimeModuleContext{
   203			Context:   runtimeCtx,
   204			VM:        vm,
   205			Loop:      loop,
   206			Owner:     owner,
   207			AddCloser: rt.AddCloser,
   208			Values:    runtimeValues,
   209		}
   210		for _, registrar := range f.runtimeModuleRegistrars {
   211			if err := registrar.RegisterRuntimeModules(moduleCtx, reg); err != nil {
   212				_ = rt.Close(ctx)
   213				return nil, fmt.Errorf("runtime module registrar %q: %w", registrar.ID(), err)
   214			}
   215		}
   216	
   217		reqMod := reg.Enable(vm)
   218		console.Enable(vm)
   219		buffer.Enable(vm)
   220		url.Enable(vm)
   221		if err := installPerformanceGlobals(vm); err != nil {
   222			_ = rt.Close(ctx)
   223			return nil, err
   224		}
   225		if err := installConsoleTimers(vm); err != nil {
   226			_ = rt.Close(ctx)
   227			return nil, err
   228		}
   229		rt.Require = reqMod
   230	
   231		initCtx := &RuntimeContext{
   232			Context: runtimeCtx,
   233			VM:      vm,
   234			Require: reqMod,
   235			Loop:    loop,
   236			Owner:   owner,
   237			Values:  rt.Values,
   238		}
   239		for _, init := range f.runtimeInitializers {
   240			if err := init.InitRuntime(initCtx); err != nil {
   241				_ = rt.Close(ctx)
   242				return nil, fmt.Errorf("runtime initializer %q: %w", init.ID(), err)
   243			}
   244		}
   245	
   246		return rt, nil

    82	// Close shuts down runtime-owned resources.
    83	func (r *Runtime) Close(ctx context.Context) error {
    84		if r == nil {
    85			return nil
    86		}
    87	
    88		var retErr error
    89		r.closeOnce.Do(func() {
    90			r.closerMu.Lock()
    91			r.closing = true
    92			closers := append([]func(context.Context) error(nil), r.closers...)
    93			r.closers = nil
    94			r.closerMu.Unlock()
    95	
    96			if r.runtimeCtxCancel != nil {
    97				r.runtimeCtxCancel()
    98			}
    99			if r.VM != nil {
   100				runtimebridge.Delete(r.VM)
   101			}
   102	
   103			for i := len(closers) - 1; i >= 0; i-- {
   104				if err := closers[i](ctx); err != nil {
   105					retErr = errors.Join(retErr, err)
   106				}
   107			}
   108			if r.Owner != nil {
   109				if err := r.Owner.Shutdown(ctx); err != nil {
   110					retErr = errors.Join(retErr, err)
   111				}
   112			}
   113			if r.Loop != nil {
   114				r.Loop.Stop()
   115			}

   115		if strings.TrimSpace(s.id) != "" {

   121	func (s namedDefaultRegistryModulesSpec) Register(reg *require.Registry) error {
   122		if reg == nil {
   123			return fmt.Errorf("require registry is nil")
   124		}
   125		for _, rawName := range s.names {
   126			name := strings.TrimSpace(rawName)
   127			if name == "" {
   128				return fmt.Errorf("default registry module name is empty")
   129			}
   130			mod := modules.GetModule(name)
   131			if mod == nil {
   132				return fmt.Errorf("default registry module %q is not registered", name)
   133			}
   134			reg.RegisterNativeModule(mod.Name(), mod.Loader)
   135		}
   136		return nil

    12	// whose `exports` property can be populated.

    10	type Scheduler interface {
    11		RunOnLoop(fn func(*goja.Runtime)) bool
    12	}

    14	// CallFunc is executed against the runtime owner context.
    15	type CallFunc func(context.Context, *goja.Runtime) (any, error)

    20	// Runner provides safe request/response and fire-and-forget execution against a goja runtime.
    21	type Runner interface {
    22		Call(ctx context.Context, op string, fn CallFunc) (any, error)
    23		Post(ctx context.Context, op string, fn PostFunc) error
    24		Shutdown(ctx context.Context) error
    25		IsClosed() bool
    26	}

    28	// Options configures a runner.
    29	type Options struct {
    30		Name          string
    31		MaxWait       int64 // milliseconds; <=0 disables implicit timeout
    32		RecoverPanics bool
    33	}
    62	func (r *runner) Call(ctx context.Context, op string, fn CallFunc) (any, error) {
    63		if r == nil || fn == nil {
    64			return nil, fmt.Errorf("runtimeowner %s: nil runner or function", op)
    65		}
    66		if r.closed.Load() {
    67			return nil, ErrClosed
    68		}
    69		ctx = normalizeContext(ctx)
    70		var cancel context.CancelFunc
    71		if r.opts.MaxWait > 0 {
    72			if _, ok := ctx.Deadline(); !ok {
    73				ctx, cancel = context.WithTimeout(ctx, time.Duration(r.opts.MaxWait)*time.Millisecond)
    74				defer cancel()
    75			}
    76		}
    77	
    78		if r.isOwnerContext(ctx) {
    79			return r.invoke(ctx, op, fn)
    80		}
    81	
    82		resultCh := make(chan callResult, 1)
    83		accepted := r.scheduler.RunOnLoop(func(vm *goja.Runtime) {
    84			ownerCtx := r.withOwnerContext(ctx)
    85			select {
    86			case <-ownerCtx.Done():
    87				resultCh <- callResult{
    88					err: fmt.Errorf("runtimeowner %s: %w: %v", op, ErrCanceled, ownerCtx.Err()),
    89				}
    90				return
    91			default:
    92			}
    93			v, err := r.invoke(ownerCtx, op, fn)
    94			resultCh <- callResult{value: v, err: err}
    95		})
    96		if !accepted {
    97			return nil, fmt.Errorf("runtimeowner %s: %w", op, ErrScheduleRejected)
    98		}
    99	
   100		select {
   101		case <-ctx.Done():
   102			return nil, fmt.Errorf("runtimeowner %s: %w: %v", op, ErrCanceled, ctx.Err())
   103		case res := <-resultCh:
   104			return res.value, res.err
   105		}
   106	}
   107	
   108	func (r *runner) Post(ctx context.Context, op string, fn PostFunc) error {
   109		if r == nil || fn == nil {
   110			return fmt.Errorf("runtimeowner %s: nil runner or function", op)
   111		}
   112		if r.closed.Load() {
   113			return ErrClosed
   114		}
   115		ctx = normalizeContext(ctx)
   116		var cancel context.CancelFunc
   117		if r.opts.MaxWait > 0 {
   118			if _, ok := ctx.Deadline(); !ok {
   119				ctx, cancel = context.WithTimeout(ctx, time.Duration(r.opts.MaxWait)*time.Millisecond)
   120			}
   121		}
   122	
   123		select {
   124		case <-ctx.Done():
   125			if cancel != nil {
   126				cancel()
   127			}
   128			return fmt.Errorf("runtimeowner %s: %w: %v", op, ErrCanceled, ctx.Err())
   129		default:
   130		}
   131	
   132		if r.isOwnerContext(ctx) {
   133			if cancel != nil {
   134				defer cancel()
   135			}
   136			r.invokePost(ctx, op, fn)
   137			return nil
   138		}
   139	
   140		accepted := r.scheduler.RunOnLoop(func(vm *goja.Runtime) {
   141			if cancel != nil {
   142				defer cancel()
   143			}
   144			ownerCtx := r.withOwnerContext(ctx)
   145			select {
   146			case <-ownerCtx.Done():
   147				return
   148			default:
   149			}
   150			r.invokePost(ownerCtx, op, fn)
   151		})
   152		if !accepted {
   153			if cancel != nil {
   154				cancel()
   155			}
   156			return fmt.Errorf("runtimeowner %s: %w", op, ErrScheduleRejected)
   157		}
   158		return nil
   159	}

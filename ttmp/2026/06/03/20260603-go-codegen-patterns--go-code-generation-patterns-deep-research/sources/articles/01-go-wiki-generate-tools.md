<article> <H2>Go Wiki: GoGenerateTools</H2>  <div> <p><a href="https://go.dev/blog/generate"><code>go generate</code></a> is only useful if you have tools to use it with! Here is an incomplete list of useful tools that generate code.</p> <ul> <li><a href="https://pkg.go.dev/golang.org/x/tools/cmd/goyacc">goyacc</a> – Yacc for Go.</li> <li><a href="https://pkg.go.dev/golang.org/x/tools/cmd/stringer">stringer</a> – Implements <code>fmt.Stringer</code> interface for enums.</li> <li><a href="https://pkg.go.dev/github.com/sourcegraph/gostringer">gostringer</a> – Implements <code>fmt.GoStringer</code> interface for enums.</li> <li><a href="https://github.com/campoy/jsonenums">jsonenums</a> – Implements <code>json.Marshaler</code> and <code>json.Unmarshaler</code> interfaces for enums.</li> <li><a href="https://pkg.go.dev/github.com/searKing/golang/tools/cmd/go-syncmap">go-syncmap</a> - Generates Go code using a package as a generic template for <code>sync.Map</code>.</li> <li><a href="https://pkg.go.dev/github.com/searKing/golang/tools/cmd/go-syncpool">go-syncpool</a> - Generates Go code using a package as a generic template for <code>sync.Pool</code>.</li> <li><a href="https://pkg.go.dev/github.com/searKing/golang/tools/cmd/go-atomicvalue">go-atomicvalue</a> - Generates Go code using a package as a generic template for <code>atomic.Value</code>.</li> <li><a href="https://pkg.go.dev/github.com/searKing/golang/tools/cmd/go-nulljson">go-nulljson</a> - Generates Go code using a package as a generic template that implements <code>database/sql.Scanner</code> and <code>database/sql/driver.Valuer</code>.</li> <li><a href="https://pkg.go.dev/github.com/searKing/golang/tools/cmd/go-enum">go-enum</a> - Generates Go code using a package as a generic template which implements interface <code>fmt.Stringer</code> | <code>binary</code> | <code>json</code> | <code>text</code> | <code>sql</code> | <code>yaml</code> for enums.</li> <li><a href="https://pkg.go.dev/github.com/alvaroloes/enumer">enumer</a> - Generates Go code that convert Go enum to/from strings.</li> <li><a href="https://pkg.go.dev/github.com/searKing/golang/tools/cmd/go-import">go-import</a> — Performs auto import of non go files.</li> <li><a href="https://github.com/ChimeraCoder/gojson">gojson</a> - Generates go struct definitions from example json documents.</li> <li><a href="https://github.com/shurcooL/vfsgen">vfsgen</a> - Generates a vfsdata.go file that statically implements the given virtual filesystem.</li> <li><a href="https://github.com/dc0d/goreuse">goreuse</a> - Generates Go code using a package as a generic template by replacing definitions.</li> <li><a href="https://4d63.com/embedfiles">embedfiles</a> - Embeds files into Go code.</li> <li><a href="https://www.colm.net/open-source/ragel/">ragel</a> - State machine compiler</li> <li><a href="https://github.com/Maratyszcza/PeachPy">peachpy</a> - x86-64 assembler embedded in Python, generates Go assembly</li> <li><a href="https://pkg.go.dev/golang.org/x/tools/cmd/bundle">bundle</a> - Bundle creates a single-source-file version of a source package suitable for inclusion in a particular target package.</li> <li><a href="https://github.com/tinylib/msgp">msgp</a> - A Go code generator for MessagePack</li> <li><a href="https://github.com/golang/protobuf">protobuf</a> - protobuf</li> <li><a href="https://github.com/thriftrw/thriftrw-go">thriftrw</a> - thrift</li> <li><a href="https://github.com/actgardner/gogen-avro">gogen-avro</a> - avro</li> <li><a href="https://github.com/dnephin/swagger-gen-types">swagger-gen-types</a> - go types from swagger specifications</li> <li><a href="https://github.com/mmcloughlin/avo">avo</a> - generate assembly code with Go</li> <li><a href="https://github.com/google/wire">Wire</a> - Compile-time Dependency Injection for Go</li> <li><a href="https://github.com/smasher164/sumgen">sumgen</a> - generate interface method implementations from sum-type declarations</li> <li><a href="https://github.com/urandom/interface-extractor">interface-extractor</a> - generates an interface of a desired type, with only methods used within the package.</li> <li><a href="https://github.com/globusdigital/deep-copy">deep-copy</a> - creates a deep copy method for the given types.</li> <li><a href="https://github.com/katef/libfsm">libfsm</a> - fsm toolkit supporting (among others) Go and Go-flavored amd64 assembly for matching regexps</li> <li><a href="https://re2c.org/index.html">re2c</a> - lexer generator for C, C++ and Go</li> <li><a href="https://gitlab.com/opennota/re2dfa">re2dfa</a> - Transform regular expressions into finite state machines and output Go source code</li> <li><a href="https://github.com/mna/pigeon">pigeon</a> - a PEG parser generator for Go</li> </ul> </div>   </article>
